# FruitsAndVegetables

Запросы SSMS:

CREATE DATABASE FruitsAndVegetables;

CREATE TABLE Products (
    Id INT PRIMARY KEY IDENTITY(1,1),
    Name NVARCHAR(100) NOT NULL,
    Type NVARCHAR(50) NOT NULL,
    Color NVARCHAR(50) NOT NULL,
    Calories INT NOT NULL
);
